Features

🎙️
100+ voices
Every macOS system voice. Alex, Samantha, Fred, plus dozens more.
🎛️
7 effects
Reverb, delay, distortion, compression, EQ, bass boost, pitch ±12st.
⚡
11 presets
Classic Tag, Trap God, Demon, Radio, Cinematic — tap and go.
💾
WAV + MP3
Lossless WAV for DAW import, MP3 for quick sharing.
🔒
No network
Runs on localhost. Nothing leaves your machine.
🪶
Zero build
Single HTML file + 120-line Flask server. No webpack.

Install (30 seconds)

# requires macOS, Python 3.9+, ffmpeg
brew install ffmpeg
git clone https://github.com/AUstnnnnnn/plugmytag-local.git
cd plugmytag-local
python3 -m venv .venv
.venv/bin/pip install flask
.venv/bin/python server.py
# open http://127.0.0.1:5173

How it works

text ─▶ macOS say ─▶ WAV ─▶ ffmpeg -af <chain> ─▶ WAV / MP3 ↑ ↑ voice reverb · delay distortion · EQ pitch · rate
